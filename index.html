Details about Real-World Experiments

For each real-world bedroom tidying tasks, we measure success by checking if the final state \( s_f \) is reached or not. For the 'sock' task, we check if the sock is successfully transported to be inside the the container box from the drawer. For the 'trash' task, we check if the trash is successfully thrown into the trash can after being grabbed off the shelf by the robot. For the 'wine' task, we check if the tissue covers the spilled wine at the end after the robot grips the tissue from the tissue box.

Details about Simulation Experiments

For the simulation tasks, we measure success by checking if the cube can be transferred to the second table from the first table. We also enforce a maximum time limit for completing the time, which is 500 time steps.
